最简单的变换

刚开始学习CSS的时候，接触到hover伪类，觉得这玩意好强啊，可以很方便地添加简单动态。

于是在一次练手中，写了这个：

<!--more-->

{.EnlighterJSRAW enlighter-language="css"} a { text-decoration: none; background-color: #000; } a:hover,.active { background-color: #f1f2f3; }
