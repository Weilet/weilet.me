安装spf13-vim

2020-03-03
Linux

spf13-vim之于vim，恰如oh-my-zsh之于zsh

背景

由于课程需要，俺需要（真正地）使用vim这一编辑器。鉴于“颜值即是第一生产力的说法”，俺是无法在黑黑丑丑的界面下写代码滴。其实在去年俺就开始折腾vim的外观配置，然而由于艺术天赋有限而艺术追求无限的缘故，最终放弃。直到前几天俺发现了spf13-vim这个好东西，当然是赶紧装上体验一番啦！

准备

spf13-vim需要vim with lua，因为它有些插件是用lua编写的。

查看是否支持lua

vim --version | if grep "+lua" &;then {echo "Vim with lua"};else {echo "Vim witho
ut lua"}

如果不支持需要安装vim-nox（安装什么取决于你的图形环境：详情）

apt-get install vim-nox

安装

根据官方文档直接安装

curl https://j.mp/spf13-vim3 -L > spf13-vim.sh && sh spf13-vim.sh

历时比较久，需要耐心等待，中间有几个插件会安装失败，属于正常行为。

安装失败的插件可以在Vundle中继续安装。在Vim的底命令模式下输入PluginInstall随后会逐个更新（或安装）预设的插件。
